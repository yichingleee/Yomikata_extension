Furigana modules

This project bundles Kuroshiro/Kuromoji assets to enable full-sentence
furigana in the popup.

Included files:
- lib/kuroshiro/kuroshiro.js (UMD)
- lib/kuroshiro/kuroshiro-analyzer-kuromoji.js (UMD)
- lib/kuroshiro/kuroshiro.esm.js (ESM wrapper around the UMD build)
- lib/kuroshiro/kuroshiro-analyzer-kuromoji.esm.js (ESM wrapper around the UMD build)
- lib/kuromoji/dict/ (dictionary files, gzipped)
